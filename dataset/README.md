This folder should contain a set of WAV files for several swearings and silence, and also has a sample of noise.

The folder strucrure is:
./<label-name>/<some-name>.wav
e.g.:
./noise.wav
./swearing-1/
  male-spoken.waw
  female-spoken.waw
  fast-spoken.waw
./swearing-2/
  male-spoken.waw
./silence/
  1.waw
  2.wav
